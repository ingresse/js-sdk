    Ingresse Ticket's API

    API to get, create, update or remove tickets.

    Example:

    var Sdk = require('ingresse-sdk');
var ingresse = new Sdk();

// You will have access to this API
// after instantiate the Sdk.
var ticket = ingresse.ticket;
